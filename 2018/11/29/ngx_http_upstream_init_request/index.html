ngx_http_upstream_init_request

概述

可以理解 upstream 是 NGINX 作为客户端主动创建的一个到后端的连接，并将连接事件添加到事件循环中。在用户调用 ngx_http_upstream_init 会创建到后端的连接，而在 ngx_http_upstream_init 中会调用 ngx_http_upstream_init_request 作为主要的逻辑处理。
