# 我的博客

地址：[juzipeek.github.io](https://juzipeek.github.io)

## 安装说明

1. 在 github 创建新项目，项目名为 [username].github.io
2. 创建好项目后可以选择导入项目作为模版，导入 http://juzipeek/juzipeek.github.io
3. `_post` 目录内容为博客内容，可以参照这个格式组织
4. `_config.yml` 中有网站域名、社交账号相关配置

## `_post` 内文章识别示例

应该在文章开头添加以下内容：

```
---
layout: post
title: 标题
category: 左侧分类
tags: 标签1，标签2
description: 详细描述
---
```
